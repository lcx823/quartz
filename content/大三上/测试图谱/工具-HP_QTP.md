# 工具-HP QTP/UFT/LoadRunner

书中提及的HP Mercury公司的测试工具集，是企业级测试解决方案的代表。这些工具后来被HP收购并整合发展，其中最著名的是QuickTest Professional (QTP) / Unified Functional Testing (UFT) 用于功能测试，以及LoadRunner用于性能测试。

## HP QTP (QuickTest Professional) / UFT (Unified Functional Testing)

QTP（后来发展为UFT）是一款强大的、商业化的自动化[[测试类型-功能测试|测试类型-功能测试]]工具，主要用于测试桌面、Web和企业级应用。

### 核心特性
- **支持多种技术**: 通过插件支持.NET, Java, SAP, Oracle等多种应用。
- **关键字视图 (Keyword View)**: 提供类似表格的视图，让不熟悉编程的测试人员也能创建和编辑测试步骤，降低了自动化门槛。
- **专家视图 (Expert View)**: 提供完整的VBScript编码环境，供高级用户编写复杂的测试逻辑。
- **对象库 (Object Repository)**: 将被测应用中的UI对象（如按钮、文本框）及其属性存储在一个独立的文件中。当UI变化时，只需更新对象库，而无需修改大量测试脚本，极大地提高了脚本的可维护性。
- **数据驱动测试 (Data-Driven Test)**: 内置DataTable功能，可以方便地将测试数据与测试逻辑分离，使用不同的数据集重复执行同一脚本。
- **检查点 (Checkpoints)**: 提供多种内置的验证点（如文本、位图、数据库检查点），用于自动比较实际结果与期望结果。
- **错误恢复机制 (Recovery Scenario)**: 预设当测试执行出错时系统应如何处理（如重启应用），以保证后续测试能继续进行。

## HP LoadRunner
LoadRunner是业界领先的商业化[[./专项测试-性能测试|专项测试-性能测试]]和[[./专项测试-负载测试|专项测试-负载测试]]工具。

### 核心组件
1.  **VuGen (Virtual User Generator)**: 虚拟用户生成器，用于录制或编写模拟用户操作的脚本。
2.  **Controller**: 控制器，用于设计、组织和监控测试场景（Scenario），如设置并发用户数、加载方式、持续时间等。
3.  **Load Generator**: 负载生成器，实际执行VuGen脚本，产生对服务器的负载。
4.  **Analysis**: 分析器，用于收集、整理和分析测试结果，生成详细的图表和报告，帮助定位性能瓶颈。

### 主要功能
- **广泛的协议支持**: 支持HTTP/HTTPS, Web Services, Oracle, SAP等多种协议，能够测试各种复杂的企业级应用。
- **真实的负载模拟**: 能够精确模拟成千上万的虚拟用户，并设置不同的[[./专项测试-负载测试#关键概念|思考时间]]和[[./专项测试-负载测试#关键概念|加载方式]]。
- **强大的监控能力**: 能实时监控应用服务器、数据库服务器、网络设备等的CPU、内存、响应时间等各项性能指标。
- **深入的分析报告**: 提供丰富的图表和数据分析功能，帮助用户深入定位性能瓶颈。
