# 青海大学研究生招生目录

检索日期：2026-05-18  
院校层级：211 / 双一流，非985  
官方入口：青海大学研究生院、青海大学研究生招生信息网

## 一、学校与招生层级

青海大学是国家“211工程”重点建设大学，入选“双一流”建设高校；官方博士简章介绍中明确，学校生态学入选“双一流”建设学科，现有7个一级学科博士学位授权点。本文仅整理研究生招生目录相关信息，不替代当年正式报名系统与学院复试细则。

## 二、2026年硕士研究生招生

### 1. 招生类型与规模

根据《青海大学2026年招收攻读硕士学位研究生招生简章》，学校2026年预计招收硕士研究生2003人（含推免生），分为学术型和专业型两类：

| 类型 | 计划信息 |
|---|---:|
| 学术型硕士 | 602人 |
| 专业型硕士 | 1401人 |
| 少数民族高层次骨干人才计划 | 26人，可接收不超过10%汉族考生 |
| 学习形式 | 普通全日制 |
| 学制 | 3年 |

最终招生人数以教育部正式下达计划为准，拟招收推免生人数以推免服务系统最终确认录取人数为准。

### 2. 硕士专业目录

官方硕士简章说明，招生专业详见《青海大学2026年硕士研究生招生专业目录》，并给出专业目录入口：

- https://yzbm.qhu.edu.cn/zsml/sszsml/index/2026

已检索到该官方目录入口，但页面为招生目录检索页，公开网页正文未直接展开完整学院、专业、方向和考试科目列表。因此本文不转写未能完整核验的逐专业表格，保留官方目录入口作为准确信息源。

可确认的硕士招生类型包括：

- 全国统考硕士研究生。
- 推荐免试硕士研究生。
- 少数民族高层次骨干人才计划硕士研究生。
- 可按政策申请少数民族照顾政策的考生类别。

### 3. 重点报考限制与政策提示

- 临床医学、中医硕士专业学位研究生招生对象为符合医师资格考试报考条件规定专业的应届或往届普通本科毕业生；已获住院医师规范化培训合格证书人员原则上不得报考临床医学、中医学专业学位研究生。
- 学校不接收定向免费培养医学生报考硕士研究生。
- “少数民族高层次骨干人才计划”须按教育部和生源地省级教育行政部门相关要求在管理平台完成报名、上传材料和签订定向协议等流程。
- 报考“少数民族高层次骨干人才计划”的考生不得调剂到该计划以外录取，未报考的不得调剂入该计划录取。

## 三、2026年推免与直接攻博

### 1. 推免硕士

根据《青海大学2026年接收优秀应届本科毕业生免试攻读研究生（含直接攻博）招生简章》，学校所有学科专业均可接收推免生；各学院接收推免生总额原则上不超过该学科专业招生计划的50%，招生学科专业以学校公布的2026年推免生招生专业目录为准。

### 2. 直接攻博

直博生面向具有推荐免试资格的优秀应届本科毕业生，学制5年。学校明确以下一级学科博士点接收直博生：

- 作物学
- 草学（含草地信息化、草业经济）
- 生态学（含生态经济、智慧生态）
- 临床医学
- 水利工程
- 化学工程与技术
- 中医学

每个专业拟接收推免生8人左右，招生学科专业以学校公布的2026年推免生招生专业目录为准。

## 四、2026年博士研究生招生

### 1. 普通博士招生

根据《青海大学招收2026年博士研究生招生简章》，学校2026年拟招收学术型全日制博士研究生297名，包含：

- 普通招考
- 硕博连读
- 直接攻博
- 工程博士
- “少数民族高层次骨干人才计划”专项计划

具体招生计划以教育部实际下达计划为准，所有学科专业均须脱产学习。

### 2. 博士招生方式

| 招生方式 | 面向对象与说明 |
|---|---|
| 普通招考 | 面向具有硕士研究生学历或学位以及符合条件的同等学力人员，采用“申请-考核”制，考试分初试和复试。 |
| 硕博连读 | 面向青海大学已完成规定课程学习和考核、成绩优秀、具有科研潜质的在学二、三年级硕士研究生，以“申请-审核”制执行。 |
| 直接攻博 | 面向优秀应届本科毕业生且具有推荐免试资格的全日制本科生；普通博士简章说明该计划中的工程博士专项计划校内6人已完成全部招生工作。 |
| 工程博士专项计划 | 第一批次报名仅接受工程博士专项计划2家企业已完成初选在职相关人员报名。 |
| 少数民族高层次骨干人才计划博士 | 除满足普通博士报考条件外，还需提供生源所在地省级教育行政部门批准的相关登记表。 |

### 3. 博士一级学科与目录情况

普通博士简章说明学校现有7个一级学科博士学位授权点；结合推免直博简章中列出的博士一级学科，2026年博士招生相关一级学科包括：

- 作物学
- 草学
- 生态学
- 临床医学
- 水利工程
- 化学工程与技术
- 中医学

官方普通博士简章提供附件《2026年博士研究生招生专业目录》（Excel），但网页检索正文未直接展开完整导师、方向、考试科目明细。本文保留官方附件链接作为完整目录来源，不转写无法完整网页核验的逐方向表格。

### 4. 博士专项计划

根据《青海大学招收2026年攻读博士学位研究生（专项计划）招生简章》，专项计划采用普通招考、“申请-考核”方式，初试科目英语、业务课一、业务课二均为免试。

专项计划招生专业和计划如下：

| 专业 | 计划 |
|---|---:|
| 水利工程（智慧能源） | 19人 |
| 化学工程与技术 | 30人 |
| 合计 | 49人 |

专项计划学制4年，学习年限4-8年；由化工学院、土木水利学院具体负责“申请-考核”选拔组织实施。

> 注：专项计划页面附件标题显示为“青海大学2025年博士研究生招生（专项计划）项目名称及招生计划一览表”，但页面标题、发布日期和正文均为2026年专项计划。本文按页面正文整理，并将附件标题不一致列为需以官方后续说明为准的待核项。

## 五、院系专业概括

### 1. 硕士层面

硕士层面可确认学校按学术型、专业型两大类别招生，学习形式为全日制，专业目录通过官方招生目录系统发布。因官方目录检索页未在公开网页正文展开完整学院专业表，本文不自行补全院系清单。

已从简章可确认的专业类别信息包括：

- 医学相关专业中，临床医学、中医硕士专业学位有明确本科专业与规培证书限制。
- 少数民族高层次骨干人才计划纳入硕士招生总计划。
- 推免生覆盖学校所有学科专业，但各学院推免接收总额原则上不超过该学科专业招生计划50%。

### 2. 博士层面

博士层面可确认的招生学院/培养组织主要围绕7个一级博士点展开；专项计划明确涉及：

- 化工学院：化学工程与技术专项计划相关组织单位。
- 土木水利学院：水利工程（智慧能源）专项计划相关组织单位。

其他普通博士招生的具体学院、导师、研究方向和考试科目以官方附件《2026年博士研究生招生专业目录》为准，本文未在网页正文中找到可完整核验的展开表。
