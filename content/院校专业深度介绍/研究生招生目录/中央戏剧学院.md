# 中央戏剧学院研究生招生目录

- 检索日期：2026-05-18
- 院校层级：双一流，非 211，非 985
- 主管部门：教育部
- 招生信息口径：以中央戏剧学院官网“招生就业 - 研究生招生”发布的 2026 年硕士、博士招生简章、专业目录、推免办法及专项计划说明为准。

## 一、招生类型概览

中央戏剧学院 2026 年研究生招生覆盖硕士研究生、博士研究生、推荐免试硕士研究生、港澳台硕士研究生、港澳台/留学生博士研究生等类型。硕士研究生基本学习年限为 3 年，学习方式为全日制；博士研究生面向全国招收全日制博士，学习年限一般为 3 年或 4 年，以专业目录为准。

硕士招生包括全国统考、接收推荐免试硕士研究生、港澳台硕士研究生。硕士学位授权点主要包括艺术学、戏剧与影视、戏曲与曲艺；其中统考硕士目录同时设置学术型和专业型方向。硕士专项计划在招生简章中明确包括“少数民族高层次骨干人才计划”和“退役大学生士兵专项计划”，并要求专项计划材料按期发送至指定邮箱。

博士招生分为学术学位博士和专业学位博士。学术学位博士招收艺术学学术学位博士研究生；专业学位博士招收戏剧与影视、戏曲与曲艺专业学位博士研究生。专业学位博士招考实行申请考核制；学术学位博士目录中既有普通考试科目设置，也有部分研究方向标注“申请-考核制”。港澳台地区考生和留学生博士报考须知另行发布。

## 二、硕士研究生招生目录概括

### 1. 招生学位类型与考试科目

2026 年硕士专业目录显示：

- 学术型硕士公共科目为 101 思想政治理论，外国语为 201 英语一、202 俄语、203 日语选一；业务课科目按各院系方向设置。
- 专业型硕士公共科目为 101 思想政治理论，外国语为 204 英语二、202 俄语、203 日语选一；业务课科目按各院系方向设置。
- 拟招生人数为初步计划，学校将根据教育部最终下达计划、学科需要、办学条件、生源情况等调整。

## 三、推荐免试硕士研究生

中央戏剧学院发布了 2026 年接收推荐免试攻读硕士学位研究生工作办法及校外推免招生目录。校外推免目录主要覆盖戏剧文学系、电影电视系、戏剧学系、数字戏剧系、实验剧团等单位，复试考核形式以面试为主。

校外推免目录列明的方向包括：

- 戏剧文学系：艺术学（戏剧学）。
- 电影电视系：艺术学（播音与主持艺术研究）；戏剧与影视（电影创作、电影创作〔剧集策划〕、电影制作技术〔录音〕、电影制作技术〔虚拟制作〕、影视制片、播音与主持艺术创作）。
- 戏剧学系：艺术学（表导演理论研究、中国传统哲学与中国演剧、中国演剧与视觉文化、戏剧美育研究、戏剧学基础）；戏剧与影视（中国演剧创新实践）。
- 数字戏剧系：艺术学（戏剧数字化研究）；戏剧与影视（戏剧数字化应用、数字戏剧导演创作）。
- 实验剧团：戏剧与影视（剧院与剧团实践）。

目录注明：拟接收推免计划未使用的，将重新纳入相应统考计划。

## 四、硕士专项计划与其他硕士招生类型

2026 年硕士招生简章设“专项计划”章节，明确：

- 少数民族高层次骨干人才计划：以考生报名时填报确认的信息为准，参加全国硕士研究生招生考试报名和统一入学考试，并按要求提交登记表和资格申请平台审核通过截图。
- 退役大学生士兵专项计划：符合教育部相关报考条件及规定者可按要求报考，并提交《入伍批准书》和《退出现役证》电子版。
- 报考专项计划的考生不再享受初试加分和少数民族照顾政策。

港澳台硕士研究生方面，学校发布了 2026 年面向香港、澳门、台湾地区招收硕士研究生简章及港澳台硕士研究生招生专业目录。港澳台硕士目录覆盖表演系、导演系、音乐剧系、京剧系、电影电视系等方向，以全日制专业型方向为主。检索到 2026 年港澳台、留学生硕士研究生招生考试工作办法及资格审核通过名单，但未检索到单独命名为“2026 年留学生硕士研究生招生简章”的官方页面；留学生硕士信息以学校后续发布的考试工作办法、资格审核名单及相关通知为准。

## 五、博士研究生招生目录概括

### 1. 学术学位博士

2026 年学术学位博士招生简章明确招收艺术学学术学位博士研究生。专业目录中的招生单位和方向包括：

| 院系/单位 | 学术博士方向概括 |
|---|---|
| 表演系 | 戏剧（影视）表演艺术研究。 |
| 导演系 | 戏剧导演理论研究。 |
| 舞台美术系 | 戏剧舞台美术研究（舞台设计、灯光设计、人物造型、舞台美术史论、剧场文化学）。 |
| 戏剧文学系 | 戏剧学、戏剧美学、中国戏曲史论、中国话剧史论、中国话剧演出史论、曲艺与戏曲。 |
| 音乐剧系 | 音乐剧表演艺术研究。 |
| 戏剧教育系 | 戏剧教育学理论研究。 |
| 戏剧管理系 | 戏剧管理研究。 |
| 电影电视系 | 电影艺术研究、电影文化研究、口语和语言传播。 |
| 戏剧学系 | 中国传统哲学与中国演剧、中国演剧体系构建研究、戏剧美学与美育。 |
| 数字戏剧系 | 戏剧与媒介研究、数字文化制度研究。 |
| 戏剧艺术研究所 | 西方戏剧、跨文化戏剧研究等方向。 |

其中戏剧学系、数字戏剧系、戏剧艺术研究所部分方向在目录中标注“申请-考核制”，学校同时发布了相应研究方向要求及说明附件。

### 2. 专业学位博士

2026 年专业学位博士招生简章明确招收戏剧与影视、戏曲与曲艺专业学位博士研究生，并说明专业学位博士生招考实行申请考核制。专业目录中的招生单位和方向包括：

| 院系/单位 | 专业博士方向概括 |
|---|---|
| 表演系 | 戏剧（影视）表演艺术创作研究；戏剧（影视）表演艺术研究（偶剧表演与设计）。 |
| 导演系 | 戏剧导演创作研究。 |
| 舞台美术系 | 戏剧舞台美术创作研究（舞台设计、数字媒体设计、人物造型、演出制作研究）。 |
| 戏剧文学系 | 编剧艺术创作研究。 |
| 音乐剧系 | 音乐剧作曲、音乐剧表导演研究。 |
| 京剧系 | 戏曲表演创作研究、戏曲导演创作研究。 |
| 戏剧教育系 | 戏剧美育理论建构与实践研究。 |
| 电影电视系 | 电影创作研究（编剧、导演、摄影、录音艺术研究）、电影创作研究（编剧、导演艺术研究）、电影创作研究（录音艺术）、媒介传播语言艺术研究。 |
| 数字戏剧系 | 传统戏剧数字化、数字戏剧创作。 |
| 实验剧团 | 剧场与剧团演出运营管理。 |
| 智能戏剧艺术空间教育部重点实验室 | 戏剧人工智能、文化科技。 |
