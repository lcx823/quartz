# 热门专业与考试类型

## 先看三种结构

| 类型 | 常见科目 | 总分 | 例子 |
|---|---|---:|---|
| 四科型 | 政治 + 外语 + 业务课一 + 业务课二 | 500 | 计算机、金融、法硕、很多工科和经管学硕 |
| 三科型 | 政治 + 外语 + 专业基础综合 | 500 | 教育学学硕、医学、历史学等 |
| 两科型 | 管理类综合能力 + 外语 | 300 | MBA、MPA、MPAcc、MEM、审计、图书情报 |

## 热门方向对比

| 方向 | 常见考试 | 最大差异 |
|---|---|---|
| 管理类联考 | 199 管综 + 外语 | 初试不考政治，政治常在复试考。不是所有管理学都考管综。 |
| 法律硕士 | 政治 + 英语 + 法硕专业基础 + 法硕综合 | 分法律（非法学）和法律（法学）。 |
| 教育学 | 学硕常见 311；专硕常见 333 + 方向课 | 先分学硕还是专硕。 |
| 医学 | 临床医学综合能力等 | 通常不考数学，但报考资格很关键。 |
| 计算机 | 数学 + 408 或院校自命题 | 专业课差异大，408 和自命题路线不同。 |
| 金融 | 数三或 396 + 431 金融学综合 | 第三科差异很关键。 |
| 会计专硕 MPAcc | 199 管综 + 英语二 | 初试看似科目少，但竞争密集，复试很重要。 |

## 最容易混淆

- 管理类联考不等于所有管理专业。
- 法硕不等于法学学硕。
- 教育学学硕和教育硕士专硕考试结构不同。
- 医学不能只看分数，还要看本科背景和职业路径。
- 计算机、金融不能只凭专业名判断科目。

## 新手判断顺序

```text
专业代码/学硕专硕
→ 初试科目结构
→ 是否考数学
→ 专业课统考还是自命题
→ 报考限制和复试内容
```
