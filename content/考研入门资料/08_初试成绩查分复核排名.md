# 初试成绩查分复核排名

## 概念速览

| 概念 | 新手解释 |
|---|---|
| 初试成绩 | 各科分数和总分。 |
| 查分 | 查询自己每科成绩和总分。 |
| 成绩复核 | 怀疑统计、登记、漏评错误时申请核查。 |
| 排名 | 学校可能给出的同专业方向内初试排序，不是全国排名。 |
| 单科线 | 每一科最低要求。 |
| 总分线 | 初试总分最低要求。 |

## 查分入口

常见查分入口有：

- 研招网成绩查询系统。
- 省教育考试院官网。
- 报考院校研究生招生网或校内系统。

研招网查分页面的数据通常由招生单位提供，招生单位负责解释。

## 成绩复核不是重新批卷

复核通常检查：

- 是否漏评。
- 是否加错分。
- 是否登错分。
- 是否统分错误。

一般不解决：

- “我觉得老师给分太低”。
- “我要看原卷”。
- “请重新批主观题”。

## 排名怎么看

有些学校会给排名，有些不会。即使给排名，也要看口径：

- 是否同一学院？
- 是否同一专业代码？
- 是否同一研究方向？
- 是否区分全日制和非全日制？
- 是否包含推免、专项计划？

排名只能参考，最终看复试线和复试名单。

## 判断顺序

1. 查各科和总分。
2. 对照国家线，看总分和单科是否都过。
3. 对照学校线、院线、专业线。
4. 等目标院校复试名单。
5. 若不稳，同步准备复试和调剂。
