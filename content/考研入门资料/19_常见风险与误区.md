# 常见风险与误区

## 核心结论

考研最大的误区不是“努力不够”，而是把考研当成单纯分数问题。实际录取由招生计划、推免占比、初试、复试、调剂规则、信息公开和时间节点共同决定。

## 十个常见误区

| 误区 | 正确理解 |
|---|---|
| 只看学校名气 | 要看具体学院、专业、方向、科目和录取数据。 |
| 拟招生人数 = 统考名额 | 很多人数包含推免、专项、非全。 |
| 过国家线就能上岸 | 国家线只是最低门槛。 |
| 复试只是走过场 | 复试不合格通常不录取。 |
| 调剂是保底 | 调剂是第二轮竞争，限制很多。 |
| 经验帖等于规则 | 规则必须回到研招网和学校官网核验。 |
| 报名成功就一定资格没问题 | 后续确认、复试、录取仍会审核资格。 |
| 只看一年数据 | 至少看近 2-3 年趋势。 |
| 多备几个完全不同专业课 | 后期换校成本极高。 |
| 初试后再准备复试 | 热门院校复试窗口很短。 |

## 最低限度核查清单

- 今年是否招生？
- 专业代码是什么？
- 学硕还是专硕？
- 全日制还是非全日制？
- 考试科目是否变化？
- 拟招生人数是多少？
- 推免占多少？
- 统考实际录取近几年是多少？
- 近几年复试线是多少？
- 拟录取最低分和中位分是多少？
- 复试占比是多少？
- 复试考什么？
- 是否接收调剂？
- 自己是否符合报考条件？
- 所有信息是否来自官方入口？

## 官方信息优先级

1. 教育部。
2. 研招网。
3. 省教育考试院。
4. 学校研究生院或研招办。
5. 学院官网。
6. 经验帖、机构资料、社群消息。
