# 术语表

| 术语 | 解释 |
|---|---|
| 考研 | 通常指全国硕士研究生招生考试。 |
| 招生单位 | 招收研究生的高校或科研机构。 |
| 学硕 | 学术学位硕士，偏研究和理论。 |
| 专硕 | 专业学位硕士，偏实践和职业能力。 |
| 全日制 | 通常脱产在校学习。 |
| 非全日制 | 通常非脱产学习，多面向在职定向人员。 |
| 定向 | 录取前签定向就业协议，毕业按协议就业。 |
| 非定向 | 毕业后自主双向选择就业。 |
| 初试 | 全国组织的笔试阶段。 |
| 复试 | 招生单位组织的进一步考核。 |
| 国家线 | 教育部公布的进入复试或调剂的初试基本要求。 |
| A区/一区 | 国家线地区分类之一，按报考单位所在地判断。 |
| B区/二区 | 国家线地区分类之一，通常分数线略低。 |
| 自划线 | 部分高校自主划定复试基本线。 |
| 院线 | 学院或专业进一步划定的复试线。 |
| 单科线 | 每个科目的最低分要求。 |
| 总分线 | 初试总分最低要求。 |
| 差额复试 | 进入复试人数多于录取名额。 |
| 调剂 | 一志愿未录取时申请其他有缺额的学校或专业。 |
| 推免 | 推荐优秀应届本科毕业生免初试攻读研究生。 |
| 保研 | 推免的通俗说法。 |
| 同等学力 | 未取得本科毕业学历但按接近本科毕业水平报考的身份。 |
| 报考点 | 负责网上确认、安排考场等的考试组织点。 |
| 拟录取 | 学校拟定录取，仍需公示、审核等。 |
| 统考 | 全国统一考试或全国统一命题科目。 |
| 自命题 | 招生单位自行命题的考试科目。 |
| 199 管综 | 管理类综合能力，管理类联考常见科目。 |
| 408 | 计算机学科专业基础，常见全国统一命题专业课。 |
| 396 | 经济类综合能力，部分经济金融类专硕使用。 |
