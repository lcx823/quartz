# 报名资格与学历限制

## 先看判断顺序

考研报名资格不是只看“我是不是本科”。正确顺序是：

1. 国家底线：教育部管理规定。
2. 学校章程：目标学校是否有附加条件。
3. 专业目录：目标专业是否限制本科专业、工作年限或学位。
4. 报考点公告：你能不能在某地报名确认和考试。
5. 专项计划文件：少干、退役士兵、强军、援藏等单独审核。

## 全国统考基本条件

| 类型 | 能不能报 | 关键限制 |
|---|---|---|
| 应届本科毕业生 | 可以 | 入学前必须取得国家承认的本科毕业证或境外学历认证。 |
| 往届本科毕业生 | 可以 | 学历通常要能在学信网核验。 |
| 专科满 2 年 | 可能可以 | 按同等学力报考，学校常有额外要求。 |
| 本科结业生 | 可能可以 | 通常按同等学力报考。 |
| 已获硕士或博士 | 可以 | 在读研究生报考通常要征得培养单位同意。 |
| 境外学历 | 可能可以 | 通常要完成教育部留学服务中心认证。 |

## 同等学力为什么要特别小心

“同等学力”不是不能考，但限制最多。常见附加条件包括：

- 通过大学英语四级或同等水平。
- 提供本科主干课程成绩。
- 发表论文或提交学术成果。
- 不得跨专业报考。
- 复试加试两门本科主干课程。
- 报名前需要研招办审核。

所以专科满 2 年或本科结业的同学，不能只看国家规定，必须逐条看目标学校招生简章和专业目录。

## 特殊专业常见限制

| 专业/类别 | 常见限制 |
|---|---|
| 法律（非法学） | 报考前所学专业通常必须是非法学。 |
| 法律（法学） | 报考前所学专业通常必须是法学。 |
| MBA、MPA、旅游管理、工程管理部分方向、教育管理等 | 通常要求本科毕业 3 年、专科或本科结业 5 年、硕博 2 年以上工作经验。 |
| 医学、口腔、临床、中医、护理等 | 常限制本科专业、执业资格、规培路径。 |
| 非全日制 | 很多学校原则上招在职定向就业人员。 |

## 报考点不是随便选

应届本科生通常选择就读学校所在地报考点。往届生一般选择工作所在地或户籍所在地报考点，但具体材料由省考试院和报考点规定，可能要求户籍、社保、居住证、工作证明。

## 自查清单

- 我的学历能否在学信网查到？
- 目标学校是否接收我的学历类型？
- 目标专业是否限制本科专业？
- 是否需要工作年限？
- 是否需要学位证或执业资格？
- 是否属于同等学力？
- 报考点需要什么材料？
- 是否申请专项计划或照顾政策？
