# 运算方法-浮点运算

浮点运算（Floating-Point Operations）用于处理[[概念-定点数与浮点数|浮点数]]，由于浮点数由阶码和尾数组成，其运算比定点运算复杂得多。

## 1. 浮点加减运算

浮点加减法是四则运算中最复杂的，必须经过以下四个步骤：

1.  **对阶 (Alignment)**
    - **目的**：使两个操作数的**阶码相等**，即小数点位置对齐。
    - **方法**：求出阶差 $\Delta j = j_x - j_y$。遵循**“小阶向大阶看齐”**的原则，将阶码小的那个数的尾数向右移位，每右移一位，阶码加1，直到阶码相等。
2.  **尾数求和 (Mantissa Addition)**
    - 将对阶后的两个尾数（通常是补码）进行定点加减运算。
3.  **规格化 (Normalization)**
    - **目的**：提高运算精度，使尾数保持在规定的范围内（如 $|S| \in [0.5, 1)$）。
    - **左规**：当尾数运算结果溢出（如 $01.xxxx$ 或 $10.xxxx$）或不满足规格化要求时，将尾数左移，阶码相应减1。
    - **右规**：当尾数运算结果溢出（如 $01.xxxx$ 或 $10.xxxx$）时，将尾数右移一位，阶码相应加1。
4.  **舍入 (Rounding)**
    - 在对阶或规格化过程中，尾数右移可能丢失低位，需要进行舍入处理以保证精度。
    - **常用方法**：**“0舍1入”**（类似于四舍五入）或**“恒置1”**（末位恒置1）。
5.  **溢出判断**：最终判断规格化后的阶码是否溢出（上溢或下溢）。

## 2. 浮点乘除运算

浮点乘除法相对简单，主要分为阶码运算和尾数运算两部分。

- **乘法**：
    - **阶码**： $j_{乘积} = j_x + j_y$
    - **尾数**： $S_{乘积} = S_x \times S_y$
- **除法**：
    - **阶码**： $j_{商} = j_x - j_y$
    - **尾数**： $S_{商} = S_x / S_y$

运算完成后，同样需要进行**规格化**和**舍入**处理。

## 3. IEEE 754 标准

现代计算机的浮点数格式和运算规则普遍遵循IEEE 754国际标准，该标准统一了浮点数的表示和运算细节，保证了跨平台计算结果的一致性。
