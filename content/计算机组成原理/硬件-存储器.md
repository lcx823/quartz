# 硬件-存储器

存储器是计算机系统中用于存放程序和数据的记忆设备，是[[框架-冯诺依曼计算机]]体系结构的核心。现代计算机系统通常采用[[框架-存储器层次结构]]来平衡速度、容量和成本。

## 核心功能

- **存放程序**：存储需要计算机执行的指令序列。
- **存放数据**：存储程序执行过程中需要处理的数据以及产生的中间和最终结果。

## 分类

存储器可以从不同角度进行分类：

- **按在计算机中的作用**：
    - **[[硬件-主存储器]] (Main Memory)**：简称主存或内存，能与[[硬件-CPU]]直接交换信息，速度快但容量相对较小。
    - **[[硬件-辅助存储器]] (Auxiliary Memory)**：简称辅存或外存，用于存放暂时不用的程序和数据，容量大但速度慢，不能与CPU直接交换信息。
    - **[[硬件-高速缓冲存储器 (Cache)]]**：位于CPU和主存之间，用于缓解两者速度不匹配的问题。
- **按存储介质**：半导体存储器、磁表面存储器、光盘存储器。
- **按存取方式**：随机存取存储器 (RAM)、只读存储器 (ROM)、顺序存取存储器。

## 工作原理与关键组件

主存储器的工作方式是**按地址存取**，即根据给定的地址对存储单元进行读或写操作。为此，主存必须配置关键的寄存器：

- **[[硬件-寄存器-MAR]] (存储器地址寄存器 Memory Address Register)**：存放欲访问的存储单元的地址。其位数决定了存储器的寻址空间大小。
- **[[硬件-寄存器-MDR]] (存储器数据寄存器 Memory Data Register)**：存放从存储单元读出或准备写入的数据。其位数等于存储字长。

> 为了能实现按地址访问的方式,主存中还必须配置两个寄存器MAR和MDR。MAR(Memory Address Register)是存储器地址寄存器,用来存放欲访问的存储单元的地址...MDR(Memory Data Register)是存储器数据寄存器,用来存放从存储体某单元取出的代码或者准备往某存储单元存入的代码...
