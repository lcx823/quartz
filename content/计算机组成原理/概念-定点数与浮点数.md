# 概念-定点数与浮点数

在计算机中，小数点的位置是隐含约定的，而不是用专门的硬件表示。根据小数点位置是否固定，数的表示方法分为定点和浮点两种。

## 1. 定点表示 (Fixed-Point Representation)
- **定义**：小数点的位置固定不变。
- **两种形式**：
    - **纯小数**：小数点约定在符号位和最高数值位之间。
    - **纯整数**：小数点约定在数值位的最末位之后。
- **特点**：
    - **优点**：运算规则简单，硬件实现成本低。
    - **缺点**：表示的数值范围有限，当处理的数不是纯小数或纯整数时，程序员需要通过比例因子进行缩放，容易导致“溢出”。

## 2. 浮点表示 (Floating-Point Representation)
- **定义**：小数点的位置是“浮动”的，由阶码来决定。
- **格式**：一个浮点数 `N` 通常表示为 `N = S × r^j`
    - **尾数 (S)**：表示数的有效数字，决定了数的**精度**。
    - **阶码 (j)**：是一个整数，用于指示小数点应浮动的位置，决定了数的**表示范围**。
    - **基数 (r)**：通常是2、8或16，是隐含约定的。
- **特点**：
    - **优点**：表示的数值范围远大于定点数，编程时不易溢出。
    - **缺点**：浮点运算过程（如对阶、规格化）比定点运算复杂，硬件实现成本更高，运算速度相对较慢。
- **规格化**：为了保证精度，浮点数的尾数通常要求是规格化数（如基数为2时，尾数最高数值位为1）。

## 比较总结

| 特性 | 定点数 | 浮点数 |
| :--- | :--- | :--- |
| **表示范围** | 有限 | 巨大 |
| **精度** | 固定 | 可变，相对精度高 |
| **溢出处理** | 容易溢出，需程序员处理 | 不易溢出，有上溢和下溢之分 |
| **运算复杂度** | 简单，速度快 | 复杂，速度慢 |
| **硬件成本** | 低 | 高 |
