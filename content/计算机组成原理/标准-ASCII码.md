# 标准-ASCII码

ASCII码（American Standard Code for Information Interchange，美国信息交换标准代码）是基于拉丁字母的一套计算机编码系统，是现代计算机中最基本、最通用的字符编码标准。

## 结构

- **基本ASCII码**：使用**7位**二进制数表示，共可表示 $2^7 = 128$ 个不同的字符。
- **扩展ASCII码**：使用**8位**二进制数（1个字节），共可表示 $2^8 = 256$ 个字符。前128个与基本ASCII码相同，后128个通常用于表示特殊符号、表格符号和带重音的字母等。

## 字符集内容

128个基本ASCII字符包括：
- **32个控制字符 (0-31)**：用于控制打印机、通信设备等，是不可打印的。例如 `CR` (回车), `LF` (换行), `BEL` (响铃)。
- **96个可打印字符 (32-127)**：
    - **空格 (SP)**
    - **10个阿拉伯数字**：'0' - '9'
    - **52个大小写英文字母**：'A' - 'Z', 'a' - 'z'
    - **33个标点和特殊符号**：如 `!`, `#`, `$`, `%`, `+`, `=` 等。

> ASCII码由 b7b6b5b4b3b2b1 这7个二进制位组成...为了提高信息传输的可靠性,通常增加一位 b8 做校验位,这样一个字符就可用8位二进制代码表示。

## 在计算机中的应用

- **文本存储与处理**：所有纯文本文件都基于ASCII码或其兼容编码（如UTF-8）存储。
- **人机交互**：键盘输入的字符被转换为ASCII码送入计算机。
- **十进制数表示**：
    - **非压缩型**：每个十进制数字用一个字节的ASCII码表示（如 '4' -> `0x34`）。
    - **压缩型**：利用ASCII码的低4位，将两个十进制数字压缩到一个字节中，这与[[标准-BCD码]]类似。
