# 硬件-运算器

运算器，又称算术逻辑单元（Arithmetic Logic Unit, ALU），是[[硬件-CPU]]的核心组成部分，主要负责执行算术运算和逻辑运算。

## 核心功能

1.  **算术运算**：执行加、减、乘、除等基本数学运算。
2.  **逻辑运算**：执行与、或、非、异或等逻辑操作。
3.  **中间结果暂存**：在运算过程中临时存放操作数和中间结果。

> 运算器用来完成算术运算和逻辑运算,并将运算的中间结果暂存在运算器内。

## 内部关键寄存器

为了完成运算任务，运算器内部通常包含一组专用寄存器：

- **[[硬件-寄存器-ACC]] (累加器 Accumulator)**：用于存放操作数以及运算后的结果（如和、差、乘积高位、被除数及余数）。
- **[[硬件-寄存器-MQ]] (乘商寄存器 Multiplier-Quotient Register)**：在乘法运算中存放乘数和乘积的低位，在除法运算中存放商。
- **[[硬件-寄存器-X]] (操作数寄存器)**：用于存放另一个操作数（如加数、减数、被乘数、除数）。

下表总结了在不同运算中各寄存器的作用：

| 运算/寄存器 | 加法 | 减法 | 乘法 | 除法 |
| :--- | :--- | :--- | :--- | :--- |
| **ACC** | 被加数及和 | 被减数及差 | 乘积高位 | 被除数及余数 |
| **MQ** | | | 乘数及乘积低位 | 商 |
| **X** | 加数 | 减数 | 被乘数 | 除数 |
