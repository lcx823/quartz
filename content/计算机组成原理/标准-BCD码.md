# 标准-BCD码

BCD码（Binary-Coded Decimal，二-十进制编码）是一种用**4位二进制数**来表示**一位十进制数**（0-9）的编码方式。

## 8421 BCD码

最常用的BCD码是**8421码**，也称为NBCD（Natural Binary Coded Decimal）。其名称来源于4个二进制位的权值分别为8、4、2、1。

| 十进制数 | 8421 BCD码 |
| :--- | :--- |
| 0 | 0000 |
| 1 | 0001 |
| 2 | 0010 |
| 3 | 0011 |
| 4 | 0100 |
| 5 | 0101 |
| 6 | 0110 |
| 7 | 0111 |
| 8 | 1000 |
| 9 | 1001 |

- **无效码**：4位二进制可以表示16种状态，但BCD码只用了其中的10种。`1010` 到 `1111` 这6个码是无效的。

## 压缩型BCD码

在计算机中，为了节省存储空间，通常将两个BCD码压缩到一个字节中存储，称为**压缩型BCD码**。
- **示例**：十进制数 `427`
    - **非压缩**：`00000100 00000010 00000111` (需要3个字节)
    - **压缩**：`0100 0010 0111` (需要1.5个字节，通常会补齐为2个字节，如 `0000 0100 0010 0111`)

## 应用

- **商业计算**：在需要精确表示十进制小数的商业和金融计算中，BCD码可以避免二进制浮点数转换带来的舍入误差。
- **数字显示**：在电子表、计算器等需要直接驱动数码管显示的设备中广泛使用。
- **与[[标准-ASCII码]]的关系**：十进制数字'0'-'9'的ASCII码的低4位正好是其8421 BCD码。
