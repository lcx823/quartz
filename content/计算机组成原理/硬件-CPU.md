# 硬件-CPU (中央处理器)

中央处理器（Central Processing Unit, CPU）是计算机的**核心部件**，负责解释和执行指令，以及处理数据。在逻辑关系和电路结构上，它通常由[[硬件-运算器]]和[[硬件-控制器]]两大部分组成。

## 核心功能

1.  **指令控制**：控制程序的顺序执行。
2.  **操作控制**：产生完成每条指令所需的控制命令。
3.  **时间控制**：对各种操作进行时间上的控制。
4.  **数据加工**：对数据进行算术和逻辑运算。
5.  **中断处理**：处理异常情况和特殊请求。

> 运算器和控制器在逻辑关系和电路结构上联系十分紧密,尤其在大规模集成电路制作工艺出现后,这两大部件往往集成在同一芯片上,因此,通常将它们合起来统称为中央处理器(Central Processing Unit, CPU)。

## 内部核心组件

- **[[硬件-运算器]] (Arithmetic Logic Unit, ALU)**：负责执行算术运算（加、减、乘、除）和逻辑运算（与、或、非、异或）。
- **[[硬件-控制器]] (Control Unit, CU)**：是计算机的指挥中心，负责从[[硬件-主存储器]]取出指令、分析指令并发出控制信号，指挥计算机各部件协调工作。
- **寄存器组 (Registers)**：CPU内部的高速存储单元，用于暂存指令、数据和地址。关键寄存器包括：
    - [[硬件-寄存器-PC]] (程序计数器)
    - [[硬件-寄存器-IR]] (指令寄存器)
    - [[硬件-寄存器-MAR]] (存储器地址寄存器)
    - [[硬件-寄存器-MDR]] (存储器数据寄存器)
    - [[硬件-寄存器-ACC]] (累加器)

## 与其他部件的关系

- CPU与[[硬件-主存储器]]和[[硬件-IO设备]]共同构成了现代计算机的三大核心部分。
- 它们之间通过[[硬件-系统总线]]进行信息交换。
- CPU与主存储器合称为**主机**。
